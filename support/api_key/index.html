Gemini API Key 발급 가이드

약 3분이면 완료됩니다

PART 1 — API 키 발급받기
STEP 1
Google에서 "aistudio" 검색

검색창에 aistudio를 입력하고
Google AI Studio 링크를 클릭하세요.

STEP 2
Get started 클릭

AI Studio 소개 페이지가 나타나면
하단의 Get started 버튼을 누르세요.

STEP 3
Google 계정으로 로그인

이메일과 비밀번호를 입력해서 로그인하세요.
이미 로그인 되어있다면 자동으로 넘어갑니다.

* Google 계정이 없으면 "계정 만들기"를 눌러 가입하세요.

STEP 4
왼쪽 상단 메뉴 버튼 클릭

Playground 화면이 보이면
왼쪽 상단 버튼을 누르세요.

STEP 5
Get API key 클릭

메뉴가 열리면 하단의
Get API key 를 클릭하세요.

STEP 6
API 키 만들기 클릭

API 키 페이지에서 오른쪽 상단
API 키 만들기 버튼을 누르세요.

STEP 7
키 만들기 클릭

팝업이 나타나면 아무것도 건드리지 말고
바로 키 만들기를 누르세요.

* 키 이름, 프로젝트 모두 기본값 그대로 OK

STEP 8
API 키 복사

키가 생성되면 세부정보 팝업이 나타납니다.
하단의 키 복사 버튼을 누르세요.

* API 키는 비밀번호와 같습니다. 절대 타인에게 공유하지 마세요!

STEP 9
앱에 붙여넣기

앱으로 돌아와서 설정 화면의
API 키 입력란에 붙여넣기 하면 끝!

PART 2 — 알아두면 좋은 것들
참고 1
API 키 관리 페이지

생성된 키는 API 키 페이지에서 언제든 확인할 수 있어요.
복사    사용량 확인

참고 2
사용량 확인하기

아이콘을 누르면
API 요청 수와 오류 수를 확인할 수 있어요.

참고 3
비율 제한 (무료 한도) 확인

메뉴 > Dashboard > 비율 제한에서
모델별 분당 요청 한도를 볼 수 있어요.

참고 4
사용 가능한 모델 확인

드롭다운을 눌러 모델별 제한을 확인하세요.
추천 모델: Gemini 2.5 Flash, Gemini 2.5 Pro
